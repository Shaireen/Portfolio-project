Theme 3 - Basic Animation

In this theme we got introduced to Javascript for the first time, we learned a lot about CSS animations and character creation. We worked further with Adobe Illustrator in order to make our first backgrounds and characters. It resulted in creating a sprite animation on the background and then a Javascript controlled animation. We had a chance to work in pairs during Project Pool and finally, develop a gamified onboarding solution for future KEA students.

Sprite animation

In this assignment we had to create our own character and background and then put it all together in a sprite animation controlled with CSS. I made a platypus inspired by Nordic mythology that is getting shot with an arrow while standing in a dark, mystical forest.

Javascript animation

This animation is based on 10 different CSS Animation, but its functionality is controlled by Javascript. Clicking on buttons makes different animationsstart and stop. Once again it involves a platypus character, this time in a more pleasant setting.

Project pool

During project pool we had a chance to work with different tasks - it was up to us which solution we want to work on. We could develop some of the advanced features or stick to the basic versions. It allowed us to improve our knowledge of Javascript and practice what we learned earlier.

Project pool day 1

I made an improved version of the colorful bubbles project we've done in pairs on first day of project pool. I have implemented some movement, as well as made new set of bubbles appear with each screen resize.

Project pool day 2

I created the a "guess the number" game. User gets to know if their guess was too high or too low, I have also implemented two of the advanced features - guesses are displayed in a list, there's also a counter for how many times the user has tried to guess the number. When guessed correctly, the alert says how many guesses it took

Onboarding app

Onboarding app was the final step in Basic Animation theme. We got introduced to the idea of gamification and had to come up with an interesting solution that would be useful for people who consider studying at KEA and moving to Denmark. In a group we created a solution that involves a main character - a detective who is trying to solve the mystery of missing sticky notes and gains knowledge about different KEA facilities along the way.
