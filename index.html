portfolio

This website is a representation of all the knowledge I gained during last 4 months. It contains the most important projects from all of four themes, as well as short descriptions and in some cases, the process of creating the final product. Each of the subpages is dedicated to one theme and includes screenshots, text and links to the original files or websites.

Theme 1 - Basic Web

In this theme we learned about the basic design principles, typography, colours... and much more. We got introduced to Adobe InDesign, Illustrator and created our first moodboard. Moreover, we started to learn HTML, CSS and how to work with it, as well as validate it. After finishing the first project, which was a mobile website, we began working with wireframes, responsive web design and creating prototypes in Adobe Xd, what resulted in creating a responsive website at the end of the theme.
Read more

Theme 2 - Basic UX

During second theme we started to learn about interaction between users and user interfaces. We got introduced to data collection and different ways of research. We worked with observation, interviews, surveys... We also learned about content production, information architecture and different useful tools connected to it, such as card sorting and concept mapping. Moreover, we talked about copy and microcopy, continued working with prototypes and participated in Design Sprint for the first time.
